# Animato

## Overview

Custom bootstrap 4 template that represents a fancy character called animato. It demonstrates Flexbox, CSS keyframe animations, CSS transitions, Responsive design (media queries), CSS custom properties.

## 🛠 Built With

**Languages:** HTML · CSS · JavaScript

**Libraries / tools:** `Bootstrap`, `jQuery`, `Animate.css`, `WOW.js`, `Google Fonts`

## 💡 Techniques Demonstrated

- Flexbox
- CSS keyframe animations
- CSS transitions
- Responsive design (media queries)
- CSS custom properties
- backdrop / filters
- Fetch / API calls
- CSS transforms & 3D
