## 💡 Techniques Demonstrated

- CSS Grid
- Flexbox
- CSS keyframe animations
- CSS transitions
- Responsive design (media queries)
- CSS custom properties
- clip-path shapes
- backdrop / filters
- CSS transforms & 3D
