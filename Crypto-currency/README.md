# Crypto-currency

> Multipages crypto currency website template using boostrap 4 & sass

### 🔗 [View Live Demo](https://fadyehabamer.github.io/web-templates/Crypto-currency/)

## Overview

Multipages crypto currency website template using boostrap 4 & sass. It demonstrates Flexbox, CSS keyframe animations, CSS transitions, Responsive design (media queries), CSS custom properties.

## 🛠 Built With

**Languages:** HTML · CSS · Sass · JavaScript

**Libraries / tools:** `Bootstrap`, `jQuery`, `Owl Carousel`, `Google Fonts`

## 💡 Techniques Demonstrated

- Flexbox
- CSS keyframe animations
- CSS transitions
- Responsive design (media queries)
- CSS custom properties
- clip-path shapes
- backdrop / filters
- Inline SVG
- Fetch / API calls
- CSS transforms & 3D
